Messages between nodes on the network follow the following specification:

Query:

    JSON:
        {"query":"BOT"}

NodeStatus:

    JSON:
        {"status":"OK"}
        {"status":"BUSY"}

Command:

    JSON:
        {"command":"<cmd>"}

Nodes:

    JSON
        ["123.45.67.89:5555", "987.654.321.0:5555"]


Heartbeat:
    
    JSON
        ["heartbeat":<timestamp when heartbeat is sent>]

HeartbeatResp:
    
    JSON
        ["heartbeat_resp":"123.45.67.89:5555"]
